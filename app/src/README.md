# Application Source Code
The `scr` directory is used for the code management of all the source code related to the building of the application.

Below is an example of a structured web application.
~~~
└── src/
   ├── HelloWorld/
   |   ├── Controllers/
   |   ├── Data/
   |   ├── Services/
   |   ├── Models/
   |   ├── ...
   |   └── HelloWorld.csproj
   └──  HelloWorld.sln
~~~
